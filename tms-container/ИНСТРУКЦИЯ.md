# TMS Container — Система расчёта контейнерных перевозок

## Быстрый старт

### 1. Получение API-ключа Яндекс.Карт

1. Перейдите на https://developer.tech.yandex.ru/
2. Создайте аккаунт / войдите
3. Нажмите «Подключить API» → выберите **JavaScript API и HTTP Геокодер**
4. Скопируйте API-ключ

### 2. Вставить ключ в два места

**Бэкенд** — файл `backend/.env`:
```
YANDEX_API_KEY=ВАШ_КЛЮЧ_ЗДЕСЬ
```

**Фронтенд** — файл `frontend/index.html`, строка 9:
```html
<script src="https://api-maps.yandex.ru/2.1/?apikey=ВАШ_КЛЮЧ&lang=ru_RU" ...>
```

### 3. Установка зависимостей

```bash
# В корневой папке
cd backend && npm install
cd ../frontend && npm install
```

Или двойной клик на `start.bat` — установит автоматически.

### 4. Запуск

**Вариант A — двойной клик:**
```
start.bat
```

**Вариант B — два терминала:**
```bash
# Терминал 1 (Backend)
cd backend
npm run dev

# Терминал 2 (Frontend)  
cd frontend
npm run dev
```

**Вариант C — одной командой (если установлен concurrently):**
```bash
# В корне проекта
npm install
npm start
```

### 5. Открыть в браузере
```
http://localhost:3000
```

## API эндпоинты

### POST /api/route/calculate
Главный эндпоинт расчёта рейса.

**Тело запроса:**
```json
{
  "pointLoading": { "lat": 55.75, "lng": 37.62, "address": "Москва, ..." },
  "pointUnloading": { "lat": 59.93, "lng": 30.32, "address": "Санкт-Петербург, ..." },
  "pointReturn": { "lat": 55.75, "lng": 37.62, "address": "Москва, ..." },
  "avoidTolls": false,
  "truckRestrictions": true,
  "preferHighways": true,
  "ratePerKm": 55,
  "fuelConsumptionPer100km": 35,
  "fuelPricePerLiter": 65,
  "platonRate": 2.84,
  "driverDailyRate": 3000,
  "otherExpenses": 0
}
```

### GET /api/geocode/suggest?q=Москва
Подсказки для автодополнения адреса.

### GET /api/route/defaults
Дефолтные параметры расчёта.
